# 📝 Practice Assignment – Overview of Software Engineering

### 1. Which of the following best describes the primary responsibility of a software engineer within a product team?

- **A.** Creating detailed product requirement documents and defining feature scope

- **B.** Monitoring production systems and reporting performance metrics to leadership

- **C.** Coordinating project timelines and managing cross-functional communication between stakeholders

- **D.** Designing, implementing, and maintaining scalable software solutions

✅ **Correct Answer:** D. **Designing, implementing, and maintaining scalable software solutions**

---

### 2. Which of the following best describes a key advantage for a business to follow the SDLC?

- **A.** The SDLC offers an overview of the process, so stakeholders know where they fit into that process.

- **B.** It standardizes documentation and process steps across all projects in the organisation.

- **C.** It tells the QA team what test cases to write.

- **D.** It allows for creativity because of its ad hoc approach to software development.

✅ **Correct Answer:** A. **The SDLC offers an overview of the process, so stakeholders know where they fit into that process.**

---

### 3. Which of the following best describes an activity that happens in the planning phase of the Software Development Life Cycle (SDLC)?

- **A.** User requirements are written.

- **B.** The coding process begins.

- **C.** Software architecture is designed.

- **D.** Bugs are identified.

✅ **Correct Answer:** A. **User requirements are written.**

---

### 4. Which of the following best describes an alpha release?

- **A.** It is the most stable release.

- **B.** It meets all functional requirements.

- **C.** It is the first functioning version of a system released to a select group of stakeholders.

- **D.** It is intended for all users.

✅ **Correct Answer:** C. **It is the first functioning version of a system released to a select group of stakeholders.**

---

### 5. Which statement best describes the User Requirements Specification (URS)?

- **A.** It contains functional, external interface, and non-functional requirements.

- **B.** It primarily contains use cases.

- **C.** It contains policy and regulation requirements.

- **D.** It can be combined with the Software Requirements Specification to form the System Requirements Specification (SysRS).

✅ **Correct Answer:** B. **It primarily contains use cases.**
