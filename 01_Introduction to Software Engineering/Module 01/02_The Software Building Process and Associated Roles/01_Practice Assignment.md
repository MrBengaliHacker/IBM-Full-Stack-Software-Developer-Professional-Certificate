# 📝 Practice Assignment – Software Testing & Software Documentation

### 1. Which type of testing ensures that recent code changes have not negatively impacted existing functionality?

- **A.** Non-functional testing

- **B.** Regression testing

- **C.** Functional testing

✅ **Correct Answer:** **B. Regression testing**

---

### 2. A development team finishes coding a new feature. They now test how this feature interacts with the database module and an external payment API to ensure proper communication and data flow between components. Which level of testing are they performing?

- **A.** Integration testing

- **B.** Acceptance testing

- **C.** System testing

- **D.** Unit testing

✅ **Correct Answer:** **A. Integration testing**

---

### 3. A development team creates a detailed step-by-step guide explaining exactly how to merge code into the main branch, including required approvals and specific internal workflow steps. Which type of documentation is this?

- **A.** User documentation

- **B.** Technical documentation

- **C.** Standard Operating Procedure (SOP)

✅ **Correct Answer:** **C. Standard Operating Procedure (SOP)**
