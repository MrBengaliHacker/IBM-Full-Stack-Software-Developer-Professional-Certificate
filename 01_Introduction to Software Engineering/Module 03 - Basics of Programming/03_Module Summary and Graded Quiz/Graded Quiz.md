# 📝 Graded Quiz – Basics of Programming

### 1. Which of the following best describes a program component, such as a stored value, method, interface, or class, by a custom-named label?

- **A.** Identifier
- **B.** Constant
- **C.** Variable
- **D.** Container

✅ **Correct Answer:** **A. Identifier**

---

### 2. A programmer wants to develop a lightweight web-based tool that should run on multiple devices without installation, and the code should be easily updated and executed directly on the user's device every time they access the tool using an interpreted language. Which option best describes the interpreted programming language?

- **A.** A smaller program, normally scripted code, that must be created and repeated every time the code is run
- **B.** Better for programs installed on the computer or device
- **C.** Available to users who have the same operating system or a device that understands the same language
- **D.** More difficult to write in, but creates faster applications

✅ **Correct Answer:** **C. Available to users who have the same operating system or a device that understands the same language**

---

### 3. Emma is developing a performance-critical desktop application. Which is an advantage of using a compiled language over an interpreted language?

- **A.** Easier to learn and use, but requires an interpreter
- **B.** Better for websites and smaller processes that need to be repeated
- **C.** Creates a smaller program, normally using scripted code
- **D.** Takes longer to write the code, but it runs faster

✅ **Correct Answer:** **D. Takes longer to write the code, but it runs faster**

---

### 4. Which statement correctly defines the primary purpose of a query language?

- **A.** To write code that interacts directly with CPU registers and memory using mnemonics
- **B.** To translate human-readable code directly into machine code instructions
- **C.** To create programs using object-oriented principles like classes and objects
- **D.** To request and manipulate data within a database system

✅ **Correct Answer:** **D. To request and manipulate data within a database system**

---

### 5. In programming fundamentals, which logic construct has exactly two possible outcomes ("true" or "false")?

- **A.** Boolean expression
- **B.** if-then-else statement
- **C.** do-while loop
- **D.** Branching statements

✅ **Correct Answer:** **A. Boolean expression**

---

### 6. A team wants to collaborate on a large project involving developers from different programming backgrounds. Which method should be used to outline the code logic?

- **A.** Use flowchart symbols and diagrams
- **B.** Write Python code for the project structure
- **C.** Write pseudocode to describe the program steps in simple language
- **D.** Start with SQL queries to structure database logic

✅ **Correct Answer:** **C. Write pseudocode to describe the program steps in simple language**

---

### 7. You are developing a cross-platform application where you want to write line-by-line code without compiling the entire program. Which type of code will be used?

- **A.** A compiled programming language
- **B.** An interpreted programming language
- **C.** A query language
- **D.** An assembly programming language

✅ **Correct Answer:** **B. An interpreted programming language**

---

### 8. Where you want to check a user's input age is 18 to allow restricted access, which programming construct is suitable?

- **A.** while
- **B.** switch
- **C.** if
- **D.** GoTo

✅ **Correct Answer:** **C. if**

### 9. Tanya is building a program where she needs to store items that automatically grow or shrink as users add or remove entries. Which container should she use?

- **A.** Constant
- **B.** Array
- **C.** Variable
- **D.** Vector

✅ **Correct Answer:** **D. Vector**

### 10. While reviewing the program, you noticed that the user information is stored in variables and manipulated through reusable code blocks. Which programming concept helps organize the code into reusable and structured parts?

- **A.** Modular programming
- **B.** Extreme programming
- **C.** Software Development Life Cycle (SDLC)
- **D.** Object-Oriented Programming (OOP)

✅ **Correct Answer:** **A. Modular programming**
