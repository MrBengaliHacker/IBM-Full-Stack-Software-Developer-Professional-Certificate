## 1) Programming Languages

- **Interpreted languages** execute source code through an interpreter.

- **Compiled languages** translate source code into executable files before execution.

- **High-level languages** are easier for humans to read and write.

- **Low-level languages** are closer to machine language and provide greater hardware control.

- **Query languages** are used to retrieve and manipulate data in databases.

- **Assembly languages** use mnemonic instructions that are translated into machine code.

## 2) Code Organization

- **Algorithms** provide step-by-step instructions for solving problems.

- **Flowcharts** visually represent the flow of an algorithm.

- **Pseudocode** describes program logic using simple, language-independent statements.

## 3) Programming Logic

- **Branching** allows programs to make decisions based on conditions.

- **Looping** repeats a sequence of instructions until a specified condition is met.

- **Boolean expressions** evaluate to either **True** or **False**.

- **Variables** store values that can change during program execution.

## 4) Programming Concepts

### Identifiers
- **Identifiers** are named references to program components.

- **Constants** store values that do not change.

- **Variables** store values that can change.

### Containers
- **Arrays** store a fixed number of elements of the same type.

- **Vectors** are dynamic arrays that automatically resize as elements are added or removed.

## 5) Functions

- **Functions** are reusable blocks of code that perform a specific task.

- **Standard library functions** are built into the programming language.

- **User-defined functions** are created by programmers.

- Functions are **defined**, **called (invoked)**, and in some languages, **declared**.

## 6) Object-Oriented Programming (OOP)

- **Object-Oriented Programming (OOP)** focuses on objects rather than functions.

- **Objects** combine data (**properties**) and behavior (**methods**).

- Procedural programming separates data and functions, while OOP packages them together.
