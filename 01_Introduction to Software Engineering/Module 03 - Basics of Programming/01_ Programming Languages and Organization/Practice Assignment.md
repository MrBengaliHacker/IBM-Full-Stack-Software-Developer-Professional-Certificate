# 📝 Practice Assignment – Programming Languages and Organization

### 1. Which one of the following is an example of a compiled programming language?

- **A.** JavaScript
- **B.** Java
- **C.** Python
- **D.** HTML

✅ **Correct Answer:** **B. Java**

---

### 2. True or False? Compiled programming languages are versatile and can be used across platforms as long as there is the correct interpreter.

- **A.** True
- **B.** False

✅ **Correct Answer:** **B. False**

---

### 3. True or False? A high-level programming language uses the common English language to make its code more understandable and to increase the speed of coding and debugging programs.

- **A.** True
- **B.** False

✅ **Correct Answer:** **A. True**

---

### 4. Which of the following statements about a flowchart is correct?

- **A.** A flowchart is a basic, high-level description of an algorithm.
- **B.** A flowchart is more efficient than using pseudocode for large programming projects.
- **C.** A flowchart is a pictorial representation of an algorithm, showing the steps of the algorithm as boxes of various shapes and colors connected by arrows.
- **D.** A flowchart helps explain exactly what each line of a program should do.

✅ **Correct Answer:** **C. A flowchart is a pictorial representation of an algorithm, showing the steps of the algorithm as boxes of various shapes and colors connected by arrows.**

---

### 5. Which one of the following statements about the SQL language is true?

- **A.** It is used to create, read, update, and delete data in a database.
- **B.** It is written in binary (0s and 1s).
- **C.** It is an assembly language.
- **D.** It is used for querying non-relational data.

✅ **Correct Answer:** **A. It is used to create, read, update, and delete data in a database.**
