# 📝 Practice Assignment – Software Architecture Patterns and Deployment Topologies

### 1. Which of the following is a characteristic of a distributed system?
- A. It focuses on the decomposition of a design into logical components.
- B. It is an individual unit of encapsulated functionality.
- C. It contains multiple services that coordinate interactions using a communication protocol such as HTTP.
- D. It is an interface for a database.

✅ **Correct Answer:** **C. It contains multiple services that coordinate interactions using a communication protocol such as HTTP.**

---

### 2. Which term is best described by the statement, "entails the ability to add behavior to a component without changing other components"?
- A. Extensibility
- B. Encapsulation
- C. Reusable
- D. Independent

✅ **Correct Answer:** **A. Extensibility**

---

### 3. True or False. A software application's architecture can be designed as both peer-to-peer and event-driven.
- A. True
- B. False

✅ **Correct Answer:** **A. True**

---

### 4. Which of the following best describes a staging environment?
- A. It is intended for general users.
- B. It is intended for the developers to use while they are actively coding the application.
- C. It replicates the production environment, but is not intended for general users.
- D. It is for the QA team to test the application's individual components.

✅ **Correct Answer:** **C. It replicates the production environment, but is not intended for general users.**

---

### 5. What does the production environment take into account that pre-production environments do not?
- A. Permissions
- B. Functional requirements
- C. Load
- D. Buggy code

✅ **Correct Answer:** **C. Load**
