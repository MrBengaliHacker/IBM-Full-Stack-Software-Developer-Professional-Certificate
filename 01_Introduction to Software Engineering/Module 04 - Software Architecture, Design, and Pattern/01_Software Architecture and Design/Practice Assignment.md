# 📝 Practice Assignment – Software Architecture and Design

### 1. Which of the following best describes software architecture?

- **A.** It is intended for the developers only.
- **B.** The choice of technology stack drives the software architecture.
- **C.** It captures implementation details.
- **D.** It serves as a blueprint for the developers.

✅ **Correct Answer:** **D. It serves as a blueprint for the developers.**

---

### 2. Which of the following should be included in the Software Design Document (SDD)?

- **A.** Test cases
- **B.** Constraints and dependencies
- **C.** Code
- **D.** Object definitions

✅ **Correct Answer:** **B. Constraints and dependencies**

---

### 3. Components in a well-structured design should be __________________.

- **A.** Loosely coupled
- **B.** Tightly constrained
- **C.** Loosely cohesive
- **D.** Tightly coupled

✅ **Correct Answer:** **A. Loosely coupled**

---

### 4. Which of the following is an advantage of using UML diagrams when designing software architecture?

- **A.** They help plan out structures and behaviors in advance of coding.
- **B.** They help facilitate communication among team members.
- **C.** They help developers navigate source code.
- **D.** All of the above

✅ **Correct Answer:** **D. All of the above**

---

### 5. What is the blueprint of an object called?

- **A.** An instance
- **B.** A property
- **C.** A method
- **D.** A class

✅ **Correct Answer:** **D. A class**
