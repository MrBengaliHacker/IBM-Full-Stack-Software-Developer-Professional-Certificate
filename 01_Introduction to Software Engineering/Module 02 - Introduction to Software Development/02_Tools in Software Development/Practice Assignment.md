# 📝 Practice Assignment – Tools in Software Development

### 1. What can a JavaScript library like jQuery do for you?

- **A.** Save data to a database.

- **B.** Debug HTML errors.

- **C.** Save development time.

- **D.** Help in version control.

✅ **Correct Answer:** **C. Save development time.**

---

### 2. True or False? Inversion of control helps you to create standardized apps.

- **A.** False

- **B.** True

✅ **Correct Answer:** **B. True**

---

### 3. Which of the following is performed by a build tool?

- **A.** Transforming source code into binaries.

- **B.** Providing code for reuse.

- **C.** Installing packages.

- **D.** Storing code.

✅ **Correct Answer:** **A. Transforming source code into binaries.**

---

### 4. Fill in the blank. Package managers make the end-user's life easy by __________________.

- **A.** Securing applications.

- **B.** Backing up data.

- **C.** Fixing errors in code automatically.

- **D.** Downloading and installing packages from repositories.

✅ **Correct Answer:** **D. Downloading and installing packages from repositories.**

---

### 5. True or False? Software stacks always consist of the following three layers: presentation, business logic, and data.

- **A.** True

- **B.** False

✅ **Correct Answer:** **B. False**
