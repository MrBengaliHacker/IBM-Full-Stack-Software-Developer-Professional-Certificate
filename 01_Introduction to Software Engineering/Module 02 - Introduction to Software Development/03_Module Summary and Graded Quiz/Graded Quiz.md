# 📝 Graded Quiz – Introduction to Software Development

### 1. Liam wants to add clickable buttons for pop-up messages on the website. Which technology should Liam use in this scenario?

- **A.** Hypertext Markup Language (HTML)

- **B.** Cascading Style Sheet (CSS)

- **C.** JavaScript

- **D.** Structured Query Language (SQL)

✅ **Correct Answer:** **C. JavaScript**

---

### 2. As a front-end developer, you are creating an online shopping website that is both visually appealing and interactive. Which combination of technologies would you primarily use to achieve this?

- **A.** HTML, Python, and SQL

- **B.** CSS, Java, and Node.js

- **C.** HTML, CSS, and JavaScript

- **D.** JavaScript, AngularJS, and Django

✅ **Correct Answer:** **C. HTML, CSS, and JavaScript**

---

### 3. In web development, how does the back-end contribute to handling dynamic content and secure transactions?

- **A.** By designing the aesthetic elements of the application

- **B.** By managing server-side operations and protecting data integrity

- **C.** By ensuring the application runs smoothly on different web browsers

- **D.** By integrating multimedia elements into the user interface

✅ **Correct Answer:** **B. By managing server-side operations and protecting data integrity**

---

### 4. Amal is developing a cloud-based service that needs to retrieve data from a back-end system. Which technology should he use to enable this communication?

- **A.** Application Programming Interfaces (APIs)

- **B.** Routes

- **C.** Styles

- **D.** Integrated Development Environments (IDEs)

✅ **Correct Answer:** **A. Application Programming Interfaces (APIs)**

---

### 5. Priya is a junior developer pairing with a senior engineer who directs the implementation but lets her write the code. Which pair programming style are they using?

- **A.** Strong style

- **B.** Driver style

- **C.** Navigator style

- **D.** Ping-pong style

✅ **Correct Answer:** **A. Strong style**

---

### 6. Alex is planning to build a dynamic user interface for a web application and is evaluating tools that provide a predefined structure and workflow for development. Which of the following development frameworks should Alex use in this scenario?

- **A.** JavaScript

- **B.** Vue.js

- **C.** Git

- **D.** Ruby

✅ **Correct Answer:** **B. Vue.js**

---

### 7. While working with the cloud app, your team needs to manage changes in the source code to avoid overwriting each other's work. Which tool will you use primarily to manage and track code changes during development?

- **A.** Version control system

- **B.** Frameworks

- **C.** Libraries

- **D.** Code editor

✅ **Correct Answer:** **A. Version control system**

---

### 8. You want to improve the internal structure of your code to make it clear and maintainable, without changing its external behavior. What technique should you apply?

- **A.** Code refactoring

- **B.** Version control

- **C.** Debugging

- **D.** Continuous integration

✅ **Correct Answer:** **A. Code refactoring**

---

### 9. Seeni is managing third-party libraries and ensuring your application always uses the correct versions of dependencies. Which tool would she use in this scenario?

- **A.** Continuous integration tool

- **B.** Package manager

- **C.** Debugger

- **D.** Build tool

✅ **Correct Answer:** **B. Package manager**

---

### 10. A student is learning about different technology combinations used for web development. Which of these represents a complete software stack?

- **A.** MEAN stack

- **B.** Networking stack

- **C.** Database stack

- **D.** Front-end stack

✅ **Correct Answer:** **A. MEAN stack**
